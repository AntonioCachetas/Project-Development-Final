# Project-Development-Final
Group Project where you can connect with others, share knowledge and get help you need. Functions- Help Requests(create help request, view help request) Posts(Create post, view posts) Account (login, register)

Register- Enter your name, email and password with register buttonand return home button. After submitting registration returns you to home page. (One registerd/ logged in new buttons appear on home page (my help requests, 
my profile, Notifications, Logout)

Create Help Request- Enter Subject and Descrition, with submit button and return home button. Once submitted redirects you to all the posts page.

My Profile- Renders list and information about you such as: Help requests (displays total requests for help) posts (displays total posts) rating (based off of previous help, Displays Name, Email, Bio, Location, Education, Skills, Reviews. Edit Profile button and Return Home button (add all the informtaion needed once you click edit profile button like bio, location education etc..)

Once logged out and logged in as new or different user- Once logged in as new user and click view help requests you view the same list as everyone else would and if you click offer help button on someones post, notification will appear saying "Help offer sent to example@gmail.com. If need help go to posts where it renders all the posts and you can clcik on the user and request there help.

Notifications- Once logged in as user you can view and render a list of all the help requested by users and all the help offered to other users, buttons will be located beneath each of them saying "accept" or "Decline". If accepted new button wil appear "Rate request" once pop up will appear asking for a rating out of 5 starts where you click what you would rate the experience and there help, aswell as a comment(optional) below it where you can add any comments about the session and if it was useful or not. Then you either of two option either cancel the rating and comment or submit the ratings and comment. Once submitted it will state "Rated".

Once completed you can go back to the home page and click the view posts page and click on other users email where it will redirect you to there profile page and it will render all the reviews and ratings and the date done that they previously got. it will also display an average rating where it calculates the average rating out of 5 based off previous experiences. 

This is mostly used as a tutor or help website where you can ask for help and help others. After accepting others or requesting others posts you can email them because the email is provided and go from there.
